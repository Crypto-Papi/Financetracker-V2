Data Retention and Disposal Policy

Effective Date: December 15, 2024
Last Reviewed: December 15, 2024

1. Purpose

This policy establishes guidelines for the retention, storage, and secure disposal of consumer data collected through the Keel personal finance application. This policy ensures compliance with applicable data privacy laws including CCPA, and requirements of third-party data providers including Plaid.

2. Scope

This policy applies to all consumer data collected, processed, and stored by Keel, including:

  • Account information (email, name, authentication data)
  • Financial data obtained through Plaid (transactions, account balances, account identifiers)
  • User-generated content (manual transactions, budgets, notes)
  • Usage and analytics data

3. Data Retention Schedule

Data Type Retention Period Justification
Account credentials Duration of account + 30 days Service provision
Plaid access tokens Until user disconnects or account deletion Bank connectivity
Transaction history Duration of account + 30 days Service provision
Account balances Duration of account + 30 days Service provision
User preferences Duration of account + 30 days Service provision
Payment/subscription data 7 years after last transaction Legal/tax requirements
Support communications 3 years Customer service

4. Data Disposal Procedures

4.1 Account Deletion

When a user requests account deletion or deletes their account:

  • All personal data is marked for deletion within 24 hours
  • Plaid access tokens are revoked immediately
  • Data is permanently deleted from primary systems within 30 days
  • Backup systems are purged within 90 days

4.2 Plaid Data Disconnection

When a user disconnects a linked financial institution:

  • Plaid access token is revoked immediately
  • Associated transaction data is retained per user preference or deleted upon request

4.3 Secure Disposal Methods

  • Database records: Permanent deletion via Firebase/Firestore deletion APIs
  • Access tokens: Immediate revocation and deletion
  • Backups: Automatic expiration per backup retention schedule

5. User Rights

Users may exercise the following rights regarding their data:

  • Access: Request a copy of their personal data
  • Correction: Request correction of inaccurate data
  • Deletion: Request deletion of their data
  • Portability: Request data in a portable format
  • Disconnection: Revoke access to linked financial accounts at any time

Requests can be submitted to: privacy@keelfinances.com

6. Security Measures

  • All data encrypted at rest using AES-256 encryption
  • All data encrypted in transit using TLS 1.2+
  • Data stored on Google Cloud Platform (Firebase) with SOC 2 Type II certification
  • Access controls and authentication required for all data access

7. Policy Review

This policy is reviewed:

  • Annually, at minimum
  • When significant changes occur to data processing activities
  • When required by regulatory changes
